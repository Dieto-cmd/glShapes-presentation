About

Library with classes that represent 2D and 3D Shapes.

Functions

  • Rendering 2D and 3D models
  • Rotation, Translation and Scaling
  • Setting chosen color
  • Texture mapping

Usage example:

  • Firstly we initiate an object (m_Square is a unique_ptr that points to an object of class glShapes::Square)
  • Constructor values are respectively (side length, positionX, positionY, positionZ, red, green, blue, alpha)
  • Then we rotate the square about 75 degrees in y axis
  • In the main loop we render an object
  • As a result we get a colored square rotated about y axis
  • Optionally we can give a path to the texture as a last argument to render a textured square
